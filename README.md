# Advanced Colonies (Minecraft mod)

The modpack for our Minecraft server.

## Player Config

1. Download and install [Prism Launcher](https://prismlauncher.org/)
2. Run Prism Launcher
3. [Drag this link to the main Prism Launcher window][modpack-download]
4. Click "OK"
5. Launch the "advanced-colonies" instance and connect to the server
6. When it asks if you want to download an external modpack, click "Download"
7. When it asks to restart Minecraft, click "Yes, close the game"
8. Launch Minecraft and connect again


[modpack-download]: https://github.com/canine-systems/advanced-colonies/releases/latest/download/advanced-colonies.mrpack
