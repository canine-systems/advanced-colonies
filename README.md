# Advanced Colonies (Minecraft mod)

The modpack for our Minecraft server.

## Player Config

1. Download and install [PolyMC](https://polymc.org/download/)
2. Run PolyMC
3. In PolyMC, click "Add Instance"
4. Click "Import from Zip"
5. Copy the URL below into the "Local file or link to direct download" textbox and click OK.
6. Launch the "advanced-colonies" instance and connect to the server
7. When it asks if you want to download an external modpack, click "Download"
8. When it asks to restart Minecraft, click "Yes, close the game"
9. Launch Minecraft and connect again

